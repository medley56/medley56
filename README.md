I'm Gavin. I work at the University of Colorado's Laboratory for Atmospheric and Space Physics (LASP) as a software engineer in the Science Data Processing team. I write data processing pipelines, mostly in the cloud these days, to process raw telemetry into science data products that researchers can use in their publications. 

I've worked on science data processing for:
- Total and Spectral Solar Irradiance System (TSIS-SIM)
- Emirates Mars Mission Ultraviolet Spectrograph (EMM-EMUS)
- Libera

I'm also the creator of [Space Packet Parser, a CCSDS telemetry packet parser based on CCSDS standards.](https://github.com/medley56/space_packet_parser)

I'm a big fan of open source software and I'm NASA TOPS certified
